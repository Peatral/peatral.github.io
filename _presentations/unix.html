UNIX

Moritz, Stian und Christof

Geschichte

  • 1965 Multics wird veröffentlicht
  • 1969 PDP-7 Port von Space Travel
  • 1969 PDP-7 Entwicklung von UNIX
  • 1971 Portierung auf PDP-11
  • 1972-73 Portierung auf C (UNIX V4)
  • 1977 BSD wird veröffentlicht
  • 1979 Letzte Open-Source Version (UNIX V7)
  • 1980er Dominierendes Betriebssystem an Universitäten
  • 1993 AT&T wird von Novell aufgekauft

Architektur

Probleme der Unix-Sicherheit

  • Zugriffsberechtigungen
  • SSH-keys
  • Verwaltung der Anmeldedaten für root-Accounts
  • Verfolgung der Benutzeraktivitäten
  • Verwaltung von an Dritte abgegebene Superuser-Anmeldedaten
  • Effektivität/Sicherheit von Jump-Servern

Einsatzbereiche

  • bis 1980'er an Universitäten
  • heute: vielfältige Verwendung von Unix-ähnlichen Systemen wie z.B.:
    • Smartphones
    • Webserver
    • Mess- und Steuergeräte
    • Haushaltsgeräte
    • WLAN-Router
