Rekursion und Fibonacci

Moritz und Christof

Aufgabe

Verfasst eine iterative Funktion, die die Fakultät einer Zahl berechnet. In Python.

Eine Lösung


			def fakultaetIterativ(n):
				wert = n
				zähler = n-1

				while (zähler > 0):
					wert *= zähler
					zähler -= 1
				
				return wert

Erkennt ihr eine Regel?

0, 1, 1, 2, 3, 5, 8, 13

Es ist eine Fibonacci-Folge

f(n) = f(n-1) + f(n-2)
n0 = 0; n1 = 1;

Definition

Eine Folge/Funktion wird in ihrer Definition noch einmal aufgerufen.

Beispiele

Fakultät


			def fakultaet(n):
			
				if(n == 1):
					return 1
				else:
					return n * fakultaet(n-1)

Kaskardenartige Rekursion


			def fibonacciWert(n):
				
				wert = 0
				
				if(n >= 2):
					wert = fibonacciWert(n - 1) + fibonacciWert(n - 2) 
				elif(n == 1):
					wert = 1
				else:
					wert = 0
				
				return wert
>>>fibonacciWert(5)
5

Zu Kaskaken

Kaskardenartig heißt es gibt eine innere Baumstruktur beim Funktionsaufruf.
D.h. bei jedem "Schritt" erhöht sich die Anzahl der Funktionsaufrufe.

Rekursion


			def fibonacciFolge(stop, folge=[0, 1]):
			
				folge.append(folge[len(folge) - 1] + folge[len(folge) - 2])
				
				if(len(folge) < stop):
					folge = fibonacciFolge(stop, folge)
					
				return folge
>>>fibonacciFolge(5)
[0,1,1,2,3]
