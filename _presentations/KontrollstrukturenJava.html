Kontroll Strukturen

Was sind denn eigentlich Kontrollstrukturen?

Alles was beeinflusst wie der Code abgearbeitet wird.

if / else

wenn -> dann -> ansonsten


if (a) {
    doA(); // Wenn a true ist
} else if (b) {
    doB(); // Wenn ansonsten b true ist
} else {
    doC(); // Alle anderen Fälle
}
		

if (a)
    doA(); // Wenn a true ist
else if (b)
    doB(); // Wenn ansonsten b true ist
else
    doC(); // Alle anderen Fälle

Ternäres if / else

if/else innerhalb einer Zeile


doX(bedingung ? optionIf : optionElse);
int test = bedingung ? bedingung2 ? 404 : 420 : 69;

Switch

Effizientes überprüfen von Fällen


int x = 10;
switch(x) {
    case 1:
        System.out.println("kommt wenn x 1 ist");
    case 2:
        System.out.println("kommt wenn x 1 oder 2 ist");
        break;
    case 3:
        System.out.println("kommt wenn x 3 ist");
        break;
    default:
        System.out.println("kommt wenn x was anderes ist");
        break;
}

Schleifen

Wiederholen Code unter bestimmten Umständen


for (int i = 0; i < 10; i++)
    doX(i);

for (int i = 0; i < 10; ++i) {
    doX(i);
}
        

while (bedingung) {
    // Solange bedingung hier nicht geändert wird 
    // liegt eine Endlos-Schleife vor!
}
        

do {
    // Die Schleife wird einmal durchlaufen
    // bevor die Bedingung überprüft wird
} while (bedingung);
